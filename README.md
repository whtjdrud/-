# 카카오톡 대화 분석기

카카오톡 대화 내보내기 텍스트 파일을 불러와서  
키워드 검색 · 날짜 필터 · 발신자 필터로 메시지를 찾아볼 수 있는 프로그램입니다.

## 다운로드 (Windows .exe)

1. 상단 **Actions** 탭 클릭
2. 가장 최근 워크플로우 클릭
3. 하단 **Artifacts** → `카카오톡대화분석기` 다운로드
4. 압축 해제 후 `카카오톡대화분석기.exe` 더블클릭

## 사용법

1. 카카오톡 채팅방 → 우상단 ≡ → **대화 내용 내보내기** → 텍스트 파일(.txt) 저장
2. 프로그램 실행 → 브라우저 자동 열림
3. txt 파일 드래그&드롭
4. 키워드 / 날짜 / 발신자로 검색
5. 결과를 TXT 또는 CSV로 내보내기
